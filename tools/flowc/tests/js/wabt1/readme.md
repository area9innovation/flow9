Running the test
----------------

To build the test:

    flowc1 nodejs=1 es6=test.js readable=1 wabt_test.flow

Executing the result:

    node test.js

Dependencies:
-------------

Following components need to be installed:
- yarn
- esbuild
- js-beautify
- typescript
- npm

Typical way to install globally:

    sudo npm install -g <component>
